# C++ Basic Prep

## How to compile

```bash
c++ -Wall -Wextra -Werror -std=c++98 
```

### class name

- **`UpperCamelCase`**

### File contain class code

- `ClassName.hpp / ClassName.h`
- `ClassName.cpp` or `ClassName.tpp`
